# Te-Weather

A modern weather application built with React, TypeScript, and Vite. Get real-time weather conditions, hourly forecasts, and 7-day predictions for any city worldwide.

## Features

- Real-time weather data via Open-Meteo API
- Auto-detects your current location on load
- Hourly and 7-day forecast
- Dynamic weather backgrounds and animations
- Weather alerts for extreme conditions
- City search with autocomplete

## Tech Stack

- React 19 + TypeScript
- Vite
- Tailwind CSS v4
- Open-Meteo API (no API key required)
- Lucide React icons

## Getting Started

```bash
npm install
npm run dev
```

## Deployment

Deployed via GitHub Actions to GitHub Pages automatically on every push to `main`.
